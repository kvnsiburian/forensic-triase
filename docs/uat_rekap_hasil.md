## Perhitungan Persentase Penerimaan

Rumus (dari lembar UAT): Persentase = (total "Ya") / (jumlah responden x jumlah butir) x 100%

- Total "Ya" = 36
- Jumlah responden x butir = 3 x 12 = 36
- **Persentase Penerimaan = 36 / 36 x 100% = 100%**
- Kategori (ambang >= 60% = Diterima) = **Diterima**

Persentase per aspek (bila lembar mengelompokkan aspek; sesuaikan pengelompokan
butir dgn lembar final 12 butir):
- Aspek A Kemudahan (P1-P5)  = 15/15 x 100% = 100%
- Aspek B Kegunaan (P6-P9)   = 12/12 x 100% = 100%
- Aspek C Relevansi (P10-P12)= 9/9  x 100% = 100%
  (Catatan: pengelompokan aspek di atas PERKIRAAN dari urutan butir; cek ke lembar
  final berapa butir per aspek, karena struktur 12 butir beda dari draf 15 butir.)
